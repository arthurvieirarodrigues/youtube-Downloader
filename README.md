# youtube-Downloader
Python application with a Tkinter interface for cutting YouTube videos or local files.
Supports both manual editing and automatic cuts (using semantic analysis of subtitles).

Download videos and subtitles directly from YouTube
Use already downloaded local files (.mp4 + .srt)
Manually edit cuts with timestamps
Automatically generate cuts based on topic changes
Export cuts as .mp4 files 
Simple graphical interface built with Tkinter
